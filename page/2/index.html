有些枯燥

因为剖面图中的楼梯尝试了好几次都没能画出来，所以最终决定用天正重画施工图。说实话，重新开始没太多意思，几乎每一次接近计划中的画图时间，我总有些想要逃离，另一方面，我常常为还没有玩够电脑而感到不快。这似乎说明了我是一个比较喜欢创造的人，而事实上，或许仅仅因为画图之类事并不是我的爱好所在而已，因为在编程的时候，我同样会重复很长时间，但仅仅因为我喜欢，即使重复也能够有忘我的感觉。

嘴馋

吃早餐的时候买了一个很久没有吃过的芝麻球；晚餐除了饭菜以外，还加了一碗八宝粥；晚上跑完步之后买了一包酱香饼，回到寝室后还加了一个面包…味道都很好，好到甚至还想要。

最近自己的确是越来越嘴馋了，不过大概就像是小时候喜欢吃零食一样，现在各种想吃或许也没什么大不了吧？不过现在的我实在不希望长得太胖，然而有时候要控制自己想吃的欲望又的确不那么容易，所以我想，实在嘴馋那就安心地吃吧，只是吃完之后要利用好这摄入的能量，做好该做的事，多跑步多运动，保持身体健康。

相信学习

注意力转移

吃完早饭之后按计划开始画图，虽然有那么点慵懒的感觉但其实也没经历什么挣扎，不知不觉就画了一段时间，期间也按计划学了一点 CSS 并进一步完善了博客主题。挺想要多花点时间去做与编程相关的事，不过谁知道花的时间多了之后，自己会不会感到厌倦呢？有时候爱好或许只是我的一种逃避的路径，我现在懂得这一点，所以该画图的时候我按计划继续画图，没有因为临时的意向而更改行程。当然也没少话时间做画图之外的其他事，但总的来说，花在画图上的时间比昨天似乎又多了一点。
